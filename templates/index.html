How To


This is a webpage template made to be used by Translators working on Live Translation.


The responsible for the live transmission opens up the Receiver page, chromakeys the green-box and adjusts it to fit their needs, then waits on the Translator operating the Sender page.


The Translator opens the Sender page and types their translation in the text area; every time the enter key is pressed, the current message is sent to the Receiver page, as well as showing up on the immediate feedback just above the type area.


This webpage template was made by @mavvos on GitHub, the project page can be found clicking here.
